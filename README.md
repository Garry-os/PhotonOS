# PhotonOS

## General
### Bootloader
It is included with a bootloader named 'osloader'. It is designed to be an UEFI bootloader with no BIOS support.
It does not depends on any headers like GNU-EFI or EDK2. It uses its own UEFI headers.
### Kernel
The kernel will be starting to develop when the osloader is finished.
